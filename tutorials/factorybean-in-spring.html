Using a FactoryBean in Spring

Written by Varun Hooda

Download Fork Star 

public class SocialEngine {
    public void sayHello() {
        System.out.println("Hello !!");
    }
}

public class SocialEngineBeanFactory implements FactoryBean {
    @Override
    public SocialEngine getObject() throws Exception {
        return new SocialEngine();
    }

    @Override
    public Class getObjectType() {
        return SocialEngine.class;
    }

    @Override
    public boolean isSingleton() {
        return false;
    }
}

Spring bean configuration.

<bean id="socialEngine" class="factory_bean.SocialEngineBeanFactory"/>
